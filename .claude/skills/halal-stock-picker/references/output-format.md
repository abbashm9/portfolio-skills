# Output Format Template

Use this template for delivering the final pick list. Adapt sections based on budget size and number of picks.

## 1. Header — Market Context (2–4 sentences)

Open with a brief market backdrop:
- Major theme of the month (e.g., AI capex inflection, GLP-1 expansion, Fed pivot)
- Biggest catalyst on the calendar (e.g., NVDA earnings May 20)
- Key tailwinds and headwinds

## 2. Pick List — Tiered by Conviction

### Tier 1 — High Conviction

For each pick:

```
**N. [Ticker] (TICKER) — $price | Sector | $market cap**
*Thesis: [one line]*

- **Valuation:** [forward P/E, fair value gap, analyst target with citation]
- **Growth:** [revenue/EPS growth, guidance, momentum in fundamentals with citation]
- **Financial strength:** [margins, FCF, debt — must include halal compliance confirmation]
- **Momentum:** [1mo/3mo/6mo returns, RSI status, technical setup with citation]
- **Catalyst:** [upcoming earnings, launches, regulatory events]
- **Risks:** [honest 1–2 line list — every pick has risks]
- **Conviction: HIGH** | Hold horizon: 3–6mo | Exit on [trigger]
```

### Tier 2 — Strong Momentum

Same structure, less depth — focus on what's driving the move.

### Watchlist (Optional)

Names to verify halal status on, or to add if positions stop out.

## 3. What I'd Skip This Month

1–3 names that look tempting but don't make the cut. Brief reason each.

## 4. Position Sizing Table

Use a clean table for any budget:

| # | Ticker | $ Alloc | % of Portfolio | Entry | Stop | TP1 | TP2 | TP3 | Catalyst |
|---|--------|---------|---------------|-------|------|-----|-----|-----|----------|
| 1 | XXXX | $260 | 36% | $X | $X (-Y%) | $X (sell 30%) | $X (sell 40%) | $X (hold 30%) | Date / event |

**Total: $XXX**

## 5. Scenario Outcomes Table

| Scenario | Portfolio Value | Return |
|----------|----------------|--------|
| All TP1 hit | $XXX | +X% |
| Half TP1, half stops | $XXX | +X% |
| All stops fire | $XXX | -X% |
| Best case (all TP3) | $XXX | +X%+ |

## 6. Universal Exit Rules Table

| Trigger | Action |
|---------|--------|
| Halal compliance breach (any name) | Sell at market open next day |
| Single position >45% of portfolio | Trim back |
| Portfolio down >8% from start | Stop trading, move to cash, re-evaluate |
| Stop-loss hit | Exit. No averaging down. |
| Major sector-specific risk event | Position-specific override (see exit rules) |

## 7. Execution Notes

- Order type recommendations (limit vs market)
- GTC stop-loss setup recommendation
- Whether to scale in over multiple days or buy all at once
- Recommended review cadence (Friday close, not daily)

## 8. Standard Disclaimer

End with:

> Re-verify halal status on Zoya or Musaffa before buying. Compliance can shift quarterly.
>
> This is research output, not financial advice. Past performance does not guarantee future results. Do your own due diligence.

## Style notes

- Use citations on every numeric claim from web search
- Don't bullet-point inside prose paragraphs; tables for data, prose for context
- Bold tickers and key numbers, not entire sentences
- Honest about risks — every position has them; saying so builds trust
- Don't oversell. If a pick is borderline, say so.
