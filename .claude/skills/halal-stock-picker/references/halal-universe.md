## Quarterly re-screening reminders

After every earnings season:

1. Pull latest 10-Q / 10-K data
2. Recalculate AAOIFI ratios using current market cap
3. Check Zoya / Musaffa for status changes
4. Drop any name that newly fails — replace with a similar compliant alternative
5. Note: companies issuing large debt for buybacks or M&A can quickly fail the 30% debt threshold
